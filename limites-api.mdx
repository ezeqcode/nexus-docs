## Qual é o limite?

Muito generoso. Você pode realizar até 1.000 request por segundo.

<Tip>Caso precise ralizar mais request do que 1.000 por segundo, ficaremos felizes em atender. Escreva para suporte@bitpix.com.br e vamos conversar sobre a liberação pra você!</Tip>
